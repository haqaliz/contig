# Implementation Plan — mag-firing

**Date:** 2026-07-11 · **Spec:** `./spec.md` · **Template:** the methylseq plan
(`../methylseq-firing/plan_20260711.md`) — same seam, same discipline.
**Execution:** strict TDD, commits per phase, `uv run pytest` green after each.

Difference from methylseq: multi-**bin** files (entity key = bin), parsers return
`{bin: {slug: float}}`.

## Phase 1 — QUAST + CheckM parsers (`mag_metrics.py`)
- RED: unit tests from inline `transposed_report.tsv` + CheckM summary strings.
- GREEN:
  - `parse_quast_report(path) -> {bin: {n50}}` — header-driven TSV, one row per
    bin/assembly, read the `N50` column (case-insensitive).
  - `parse_checkm_summary(path) -> {bin: {completeness, contamination}}` — read
    `Bin Id`, `Completeness`, `Contamination` columns.
  - Floor: absent/non-numeric omitted; unrecognized file → `{}`.
- Commit: `test+feat(verify): QUAST + CheckM parsers for mag metrics [C3]`

## Phase 2 — Runner gate + M6
- RED: gate tests (C1–C7) in `test_run_qc.py`, mirroring the methylseq gate tests.
- GREEN in `runner.py`:
  - `_locate_mag_qc(run_dir) -> {bin: {slug: float}}` — rglob
    `*transposed_report*.tsv` and the CheckM summary, route to parsers, **merge by
    bin** (`setdefault(bin, {}).update(...)`).
  - Gate `if assay == "mag":` mirroring the methylseq gate exactly, with
    `check=f"mag_qc:{bin}"`, `MAG_RULE_PACK`.
  - Add `"mag"` to `_DEDICATED_METRIC_ASSAYS`.
  - Imports: `mag_metrics` parsers + `MAG_RULE_PACK`.
- Validation: `uv run pytest -q` green (incl.
  `test_rule_pack_for_mag_returns_the_mag_pack`).
- Commit: `feat(verify): mag QC gate fires from QUAST/CheckM; single-source [C3]`

- Commit: `test+docs(verify): realistic QUAST/CheckM fixture + mag firing changelog [C3]`

## Edge cases
Multi-bin (C5); partial (QUAST only → completeness/contamination absent, no false
UNVERIFIED, C4); zero-usable-metrics bin → explicit `mag_qc:<bin>` UNVERIFIED (C3);
contamination is upper-bound (`warn_above`/`fail_above`); no double-emit (C6). Do
not touch other assays; CheckM only (BUSCO deferred).
