## Edge cases & error handling

- Empty/header-only VCF → `variant_count == 0` → WARN below band (never a false pass, never
  unverified). No VCF at all → gate never fires (structural QC owns a missing output).
- Duplicated site lines → counted once (documented; `parse_vcf` dedups by site key).
- Multiallelic record → one site (comma-ALT key); consistent with `parse_vcf`.
- `QCResult.value` int-vs-float: verify `models.QCResult.value` accepts an int before Phase 2
  GREEN; if strictly `float`, prefer widening the annotation to `float | int | None` over
  lossily casting the count.
