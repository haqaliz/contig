## 5. Environment & Reproducibility

- No env vars, no external tools, no network. `uv sync` already done in the worktree.
- Determinism: scenarios are data; timestamps only stamped at `--update-baseline`
  (passed in, never `Date.now()` in the scored path); `corpus_sha` binds the baseline
  to the exact scenarios file, exactly as the detector holdout binds its corpus.

## 6. Edge Cases & Error Handling

- **Trace-less failure** (`PipelineExecutionError(1, None)`): caught in the driver →
  `recovered=False`, `actual_outcome="no_record"`, never a crash (transcribe
  `test_self_heal.py:128-137`).
- **Loop retries beyond declared attempts:** scripted executor clamps to the last
  `AttemptSpec` (so a mis-declared scenario surfaces as a mismatch, not an IndexError).
- **Harness-bug vs regression:** with outcome-match as the headline, any scenario not
  reaching its declared outcome simply lowers the rate → `REGRESSION` naming it; no
  separate code path needed.
- **Missing baseline / missing scenarios file:** exit 1 with a nudge (mirror
  eval-guard).
- **`sha_mismatch`:** loud non-failing warning (set changed, baseline not refreshed).
