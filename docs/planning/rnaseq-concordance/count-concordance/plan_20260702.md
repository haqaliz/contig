## 2. Module contract (target API — write tests against this)

`src/contig/verification/count_concordance.py`:

```python
# constants
_COUNT_CONCORDANCE_ASSAYS = {"rnaseq"}
_SPEARMAN_WARN_BELOW = 0.90
_FRACTION_AGREEING_WARN_BELOW = 0.90
_AGREEMENT_REL_TOL = 0.10          # a gene "agrees" when |a-b|/max(a,b,1) <= 0.10
_MIN_SHARED_GENES = 10             # below this, correlation is UNVERIFIED (never PASS)
_COUNT_MATRIX_GLOB = "*salmon.merged.gene_counts*"   # exported for the CLI resolver

def parse_count_matrix(path) -> dict[str, float]:
    """gene_id -> summed count across all numeric columns. gzip-transparent.
    Skips any row with no parseable numeric column (this drops the header row).
    A repeated gene_id ACCUMULATES (sum), never last-wins."""

def _spearman(xs: list[float], ys: list[float]) -> float | None:
    """Hand-rolled Spearman rho: average-rank tie handling, then Pearson of ranks.
    None when < 2 points or when either ranked vector has zero variance."""

@dataclass(frozen=True)
class CountConcordanceStats:
    shared: int
    rho: float | None
    fraction_agreeing: float | None
    overlap: float            # |A∩B| / |A∪B|, 0.0 when union empty

def count_concordance(matrix_a, matrix_b) -> CountConcordanceStats: ...

def concordance_results(matrix_a, matrix_b) -> list[QCResult]:
    """Emit spearman_concordance (WARN<0.90), fraction_agreeing (WARN<0.90),
    gene_overlap (informational, ALWAYS pass). UNVERIFIED (value=None) for the two
    WARN-capped checks when shared < _MIN_SHARED_GENES. All kind='concordance'."""

def evaluate_count_concordance(primary, second, assay) -> list[QCResult]:
    """[] unless assay in _COUNT_CONCORDANCE_ASSAYS; else concordance_results(...)."""
```

QCResult factory: reuse the germline pattern — a local `_concordance(check, status,
message, value=None, expected_range=None)` that hardcodes `kind="concordance"`
(copy the shape from `concordance.py:43-58`). Only ever emit statuses
`pass|warn|unverified` so `overall_verdict` (`models.py:78-96`) can never let
concordance FAIL a verdict.

**Spearman detail (hand-rolled, deterministic):**
1. `rank(vals)`: sort indices by value; assign average rank to ties (1-based).
2. rho = Pearson correlation of `rank(xs)` and `rank(ys)`.
3. Guard: return `None` if `len < 2` or either rank vector is constant (zero
   variance → correlation undefined). A `None` rho → the spearman check is
   UNVERIFIED (never PASS).

**fraction_agreeing detail:** over shared genes, a gene agrees when
`abs(a-b) / max(a, b, 1) <= _AGREEMENT_REL_TOL`. Fraction = agreeing / shared. This
makes all-zero genes agree (diff 0) and never divides by zero.

**gene_overlap detail:** `overlap = |shared| / |union|` (0.0 if union empty), always
`status="pass"`, `message` frames it as informational context.

## 7. Edge Cases & Error Handling (all become tests)

- Header row (no numeric cols) → skipped, not a phantom gene.
- `gene_name`/label columns interleaved with numeric → only numeric summed.
- Duplicate gene id → summed.
- All-zero genes → agree (`|a-b|/max(a,b,1)=0`), no ZeroDivision.
- Tiny counts (1 vs 2) → disagree, no crash.
- `< 10` or `0` shared genes → UNVERIFIED (never PASS); overlap still reported.
- Constant/degenerate vector → Spearman None → UNVERIFIED.
- Missing primary matrix / non-rnaseq assay → clear CLI skip note, exit unchanged.
- `.tsv.gz` → gzip-transparent parse.
