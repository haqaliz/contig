## Phase 1 — The STARsolo quantifier seam

- **Goal:** a pure, unit-tested `sc_count_quantifier.py` mirroring the kallisto seam, with the
  STARsolo argv/read-order/chemistry logic fully covered and the subprocess kept out of CI.
- **Prerequisites:** baseline green.
- **Files:** create `src/contig/verification/sc_count_quantifier.py` +
  `tests/verification/test_sc_count_quantifier.py`.
- **Implementation steps (TDD — write each test first, RED, then GREEN):**
  1. `ScCountQuantifier = Callable[[str, str, str, str, str], str]` — signature
     `(reads, index, whitelist, chemistry, out_dir) -> produced matrix.mtx path`. Docstring:
     "the CLI auto path takes one of these; tests inject a fake."
  2. `SecondScQuantifierError(Exception)` — one named error for every unrunnable path (missing
     binary / reads / index / whitelist, nonzero exit, missing Solo `matrix.mtx`). Never a
     silent/partial matrix.
  3. `_CHEMISTRY_PRESETS: dict[str, tuple[int,int,int,int]]` = `{"10xv3": (1,16,17,12)}` as
     `(CBstart, CBlen, UMIstart, UMIlen)`; optionally add `"10xv2": (1,16,17,10)`. A helper
     `chemistry_geometry(name)` returns the tuple or raises `SecondScQuantifierError` on an
     unknown key. **Test AC3.**
  4. **`readfiles_order(fastqs: list[str]) -> tuple[str, str]`** — the footgun. The sample sheet
     yields `[fastq_1(=CB/R1), fastq_2(=cDNA/R2), …]`; STARsolo wants `(cDNA, CB)`, so return
     `(fastqs[1], fastqs[0])`. Raise `SecondScQuantifierError` if fewer than 2 FASTQs (single-end
     can't be a 10x pair). **Test AC2** with a 2-element and a >2-element (first pair) list, and
     the too-few case.
  5. **`starsolo_command(fastqs, index, whitelist, chemistry, out_dir) -> list[str]`** — pure
     argv: `STAR --runMode alignReads --soloType CB_UMI_Simple --genomeDir <index>
     --soloCBwhitelist <whitelist> --soloCBstart 1 --soloCBlen 16 --soloUMIstart 17 --soloUMIlen
     12 --readFilesIn <cDNA> <CB> --soloFeatures Gene --outSAMtype None --outFileNamePrefix
     <out_dir>/`. Use `chemistry_geometry` + `readfiles_order`. Constant `_STAR = "STAR"`.
     **Test AC1**: assert token positions/values **without executing**.
  6. `tx-not-needed`: STARsolo emits gene-level counts natively — **no** t2g collapse here
     (unlike kallisto). The returned path is the Solo `matrix.mtx`; the CLI feeds it to
     `load_sc_matrix` unchanged.
  7. `run_starsolo_quantifier(reads, index, whitelist, chemistry, out_dir) -> str` — the default
     runner: validate `reads`/`index`/`whitelist` exist (else `SecondScQuantifierError`); derive
     fastqs via `fastq_paths(reads)` folding `ValueError`/`ValidationError` →
     `SecondScQuantifierError`; build argv; `subprocess.run(...)` (mark the success line
     "intentionally NOT exercised in CI"); after the run, locate the Solo `matrix.mtx` under
     `out_dir` (e.g. `next(Path(out_dir).rglob("matrix.mtx*"), None)`) → `SecondScQuantifierError`
     if absent; return its path. **Test AC4** error paths by monkeypatching `_STAR` to a bogus
     binary and by missing-input args (real `tmp_path`, STAR never runs).
- **Validation:** `uv run pytest tests/verification/test_sc_count_quantifier.py -q` green; STAR
  never invoked.
- **Commit:** `feat(verify): STARsolo second-quantifier seam for single-cell concordance autorun`

## Phase 2 — CLI wiring in `verify`

- **Goal:** `--concordance-sc-counts-auto` runs the seam and feeds the shipped evaluator, with
  the full honest contract and 6-flag mutual exclusion.
- **Prerequisites:** Phase 1 merged (dispatch imports its runner + error).
- **Files:** modify `src/contig/cli.py`, `tests/test_cli.py`.
- **Implementation steps (TDD):**
  1. Import at `cli.py` top: `run_starsolo_quantifier`, `SecondScQuantifierError` from
     `contig.verification.sc_count_quantifier`.
  2. Add flags to the `verify` signature: `concordance_sc_counts_auto: bool = typer.Option(False,
     "--concordance-sc-counts-auto", help="Auto-run a second single-cell quantifier (STARsolo)
     and corroborate the run's own matrix.")`, `whitelist: str = typer.Option(None,
     "--whitelist", help="Barcode whitelist for --concordance-sc-counts-auto (STARsolo).")`,
     `chemistry: str = typer.Option("10xv3", "--chemistry", help="Single-cell chemistry preset
     for --concordance-sc-counts-auto (default 10xv3).")`. **Reuse** existing `--reads`/`--index`;
     generalize `--index` help to name both kallisto-index and STAR-genome-dir uses (**R3a**).
  3. **Mutual exclusion (R4):** add `concordance_sc_counts_auto` to the `sum(bool(x) for x in (...))`
     tuple at `cli.py:841-859` and to the error-message flag listing. **Test AC8.**
  4. Add the dispatch branch in the `if/elif` (`:869-886`): `elif concordance_sc_counts_auto:
     concordance = _evaluate_run_sc_counts_concordance_auto(record, runs_dir, run_id, reads,
     index, whitelist, chemistry)`.
  5. New helper `_evaluate_run_sc_counts_concordance_auto(record, runs_dir, run_id, reads, index,
     whitelist, chemistry, quantifier=None)` — order matters (**R3/AC7**):
     a. `primary = _resolve_primary_sc_matrix(...)`; `if primary is None: return []` (skip note
        already printed; **do NOT spawn STARsolo**).
     b. Validate `reads`/`index`/`whitelist` given + existing; else `typer.echo("Skipping
        concordance: ...")` and `return []`.
     c. `run_q = quantifier if quantifier is not None else run_starsolo_quantifier`.
     d. `with tempfile.TemporaryDirectory() as out_dir:` → `try: second = run_q(reads, index,
        whitelist, chemistry, out_dir) except SecondScQuantifierError as e: echo skip; return []`.
     e. `return evaluate_sc_count_concordance(str(primary), second, assay="scrnaseq")`.
  6. Ensure the branch's result rides the existing off-critical-path merge (`:895-896`,
     `:913-914`) so it can never touch `ok`/exit.
- **Validation:** `uv run pytest tests/test_cli.py -q` + full `uv run pytest -q` green.
- **Tests to add in `tests/test_cli.py`** (copy the `:2061-2270` kallisto pattern; fake writes a

## Environment & Reproducibility

- No env vars, no new external service pinned in CI. STARsolo version is not pinned (tool behind
  the seam); the argv is version-tolerant and unit-fixed.
- Additive-only: no `RunRecord`/reproduce-contract/exit-code change; concordance stays off the
  critical path.

## Edge Cases & Error Handling

- Single-end / <2 FASTQs per row → `SecondScQuantifierError` (can't form a 10x cDNA,CB pair).
- Unknown `--chemistry` → `SecondScQuantifierError` (never a guessed geometry).
- Wrong chemistry passed → STARsolo calls ~no cells → few shared genes → honest UNVERIFIED.
- Primary matrix absent while inputs valid → skip **before** spawning STARsolo.
- STARsolo runs but writes no `matrix.mtx` → `SecondScQuantifierError` → skip, never a false pass.
