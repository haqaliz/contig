## Locked decisions (from the interview + dig)

- **Three checks:** `exonic_fraction` (warn low), `intronic_fraction` (warn high),
  `unassigned_fraction` (warn high). All WARN-capped, per-sample.
- **Metrics are fractions in [0,1]** (unlike the 0–100 percent-scale MultiQC packs); the
  pack bands are therefore also fractions — self-consistent within this pack.
- **Denominators (intentional, must be code-commented):** exonic/intronic over
  **Total Assigned Tags**; unassigned = `(Total Tags − Total Assigned Tags) / Total Tags`.
  The nested TSS_up_*/TES_down_* windows are **never summed** (they double-count).
- **Dedicated RSeQC-artifact parser** (not the MultiQC path — the fractions aren't in
  general-stats, verified against a real `multiqc_data.json`). Additive gate; `rnaseq`
  stays OUT of `_DEDICATED_METRIC_ASSAYS`.
- **Lenient uncalibrated defaults:** `exonic_fraction warn_below 0.50`,
  `intronic_fraction warn_above 0.30`, `unassigned_fraction warn_above 0.30`. Documented
  as engineering defaults; verified to PASS the healthy yeast test run
  (exonic≈0.9998, intronic≈0.0002, unassigned≈0.112).

2. **GREEN** — implement:
   ```python
   """Parse RSeQC read_distribution.txt into read-composition fractions.

   nf-core/rnaseq runs RSeQC `read_distribution` by default and writes
   `<sample>.read_distribution.txt`; the exonic/intronic breakdown does NOT reach
   Contig's MultiQC general-stats ingest, so this parses the artifact directly.
   Local, deterministic, stdlib-only, omit-never-guess.
   """
   from pathlib import Path

   EXONIC_FRACTION = "exonic_fraction"
   INTRONIC_FRACTION = "intronic_fraction"
   UNASSIGNED_FRACTION = "unassigned_fraction"

   _EXON_GROUPS = ("CDS_Exons", "5'UTR_Exons", "3'UTR_Exons")
   _INTRON_GROUP = "Introns"


   def parse_read_distribution(path: Path) -> dict[str, float]:
       total_tags = assigned = None
       tag_counts: dict[str, float] = {}
       for raw in Path(path).read_text().splitlines():
           line = raw.strip()
           if not line or line.startswith("="):
               continue
           # preamble: "Total Assigned Tags   129802"
           low = line.lower()
           if low.startswith("total assigned tags"):
               assigned = _trailing_number(line)
               continue
           if low.startswith("total tags"):
               total_tags = _trailing_number(line)
               continue
           if low.startswith("total reads"):
               continue
           # table rows: "CDS_Exons   146030   129779   888.71"
           parts = line.split()
           if len(parts) >= 3 and parts[0] in (*_EXON_GROUPS, _INTRON_GROUP):
               val = _to_float(parts[2])  # Tag_count column
               if val is not None:
                   tag_counts[parts[0]] = val

       out: dict[str, float] = {}
       # exonic / intronic share OF ASSIGNED TAGS (intentional; do NOT switch to Total Tags)
       if assigned is not None and assigned > 0:
           exon_present = [tag_counts[g] for g in _EXON_GROUPS if g in tag_counts]
           if exon_present:
               out[EXONIC_FRACTION] = sum(exon_present) / assigned
           if _INTRON_GROUP in tag_counts:
               out[INTRONIC_FRACTION] = tag_counts[_INTRON_GROUP] / assigned
       # unassigned share OF ALL TAGS (intentional different denominator)
       if total_tags is not None and total_tags > 0 and assigned is not None:
           unassigned = total_tags - assigned
           if unassigned >= 0:
               out[UNASSIGNED_FRACTION] = unassigned / total_tags
       return out
   ```
   with small `_trailing_number` (last whitespace token → float or None) and `_to_float`
   helpers. Keep the "denominators are intentional" comments — they satisfy the AC and the
   self-critique gap. GREEN the suite.
3. **REFACTOR** — dedupe the two number helpers if they converge; keep it pure.

     def _locate_rnaseq_composition_qc(run_dir: Path) -> dict[str, dict[str, float]]:

         One file per sample. When a sample resolves to multiple files (a run keeps
         both a published `results/` copy and an intermediate `work/` copy), prefer
         the published tree so the verdict never reads a pre-final `work/` write; a
         located-but-unparseable file still yields an empty dict so the gate can emit
         an explicit UNVERIFIED.
         """
         by_sample: dict[str, list[Path]] = {}
         for path in sorted(run_dir.rglob("*.read_distribution.txt")):
             sample = path.name[: -len(".read_distribution.txt")]
             by_sample.setdefault(sample, []).append(path)
         located: dict[str, dict[str, float]] = {}
         for sample, paths in by_sample.items():
             preferred = next(
                 (p for p in paths if "results" in {q.lower() for q in p.relative_to(run_dir).parts}),

             )
             located[sample] = parse_read_distribution(preferred)
         return located

## Edge cases & error handling

- Zero/absent `Total Assigned Tags` → exonic/intronic omitted (no ZeroDivision).
- Zero/absent `Total Tags`, or `assigned > total_tags` → unassigned omitted (no negative).
- Missing individual Group rows → that metric omitted; a subset still computes.
- Located-but-unparseable file → one `rnaseq_composition_qc:<sample>` UNVERIFIED.
- No artifact → silent skip (structural QC owns genuinely-missing outputs; read_distribution
  is NOT added to the rnaseq structural manifest).
- Duplicate `results/`+`work/` copies → published-tree preference, one result per sample.
