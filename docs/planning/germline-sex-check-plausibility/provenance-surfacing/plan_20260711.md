## Data model

`SexInference` (pydantic `BaseModel` in `models.py`), a serializable mirror of the
inference-core `SexSignals` dataclass:
```python
class SexInference(BaseModel):
    inferred_sex: str            # "XY" | "XX" | "discordant" | "indeterminate"
    x_het_ratio: float | None = None
    x_sites: int = 0
    y_variant_count: int = 0
    par_masked: bool = False
    reference_build: str | None = None
```
`RunRecord`: add `sex_inference: SexInference | None = None`.

## Environment & reproducibility

- No new dependency. `SexInference` round-trips through the bundle; legacy bundles load with
  `None`. Deterministic (same VCF → same inference).

## Edge cases & error handling

- No germline VCF at finalize → `sex_inference None`; methods/HTML omit the section.
- Indeterminate inference → captured with `inferred_sex="indeterminate"`; methods renders
  "undetermined" (no fabricated call).
- Non-germline assay → never captured (gate).
- Legacy bundle (no key) → loads, `None`, reproduces.
