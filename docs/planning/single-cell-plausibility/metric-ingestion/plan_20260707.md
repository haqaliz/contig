## Phase 2 — STARsolo `Summary.csv` parser (M1)

- **Goal:** `parse_starsolo_summary(path) -> dict[str, float]` (slug→value for one sample).
- **Files:** `src/contig/verification/scrnaseq_metrics.py` (new),
  `tests/verification/test_scrnaseq_metrics.py` (new).
- **Format:** headerless two-column CSV, `Field,Value` per row. Map (case-insensitive,
  tolerate `Gene`/`Genes`):
  - `Estimated Number of Cells` → `estimated_cells`
  - `Median Gene per Cell` (or `Median Genes per Cell`) → `median_genes_per_cell`
  - `Fraction of Unique Reads in Cells` → `fraction_reads_in_cells` (already 0–1)
  Unmapped rows ignored; non-numeric values skipped (never guessed).
- **RED:** test a synthetic Summary.csv string → returns exactly the three slugs with
  float values; a file missing a field → that slug absent (not 0); a garbage value → skipped.
- **GREEN:** implement with `csv.reader` + a `_STARSOLO_FIELD_MAP` dict and a
  `_to_float` helper (returns `None` on non-numeric).
- **Validation:** `uv run pytest tests/verification/test_scrnaseq_metrics.py -q` green.
- **Commit:** `feat(verify): STARsolo Summary.csv cell-QC parser [M1]`

## Phase 3 — Cell Ranger `metrics_summary.csv` parser (M2)

- **Goal:** `parse_cellranger_metrics(path) -> dict[str, float]`.
- **Files:** same module + test file (independent test functions from Phase 2).
- **Format:** two-line CSV — quoted header row + quoted value row. Values carry
  **comma-thousands** (`"5,000"`) and **percent** (`"92.3%"`). Map:
  - `Estimated Number of Cells` → `estimated_cells` (strip commas → `5000.0`)
  - `Median Genes per Cell` → `median_genes_per_cell`
  - `Fraction Reads in Cells` → `fraction_reads_in_cells` (**`"92.3%"` → `0.923`**, so the
    unit matches the pack band `warn_below 0.7`)
- **RED:** test `"5,000"`→5000.0, `"92.3%"`→0.923, header/value zip mapping, missing
  column → slug absent. Assert the fraction unit explicitly (0.923, not 92.3) — the unit
  collision is the highest-value test here.
- **GREEN:** `_parse_number(s)` strips `,` and trailing `%` (dividing by 100 for the
  fraction slug); a `_CELLRANGER_FIELD_MAP` maps header labels → (slug, unit). Use
  `csv.reader` to respect quoting.
- **Validation:** `uv run pytest tests/verification/test_scrnaseq_metrics.py -q` green.
- **Commit:** `feat(verify): Cell Ranger metrics_summary.csv parser w/ unit normalization [M2]`

## Phase 5 — Wire the dedicated scrnaseq gate into `_discover_qc` (M4) — depends on 1–4

- **Goal:** a finished scrnaseq run's cell-QC reaches the verdict via `SCRNASEQ_RULE_PACK`.
- **Files:** `src/contig/runner.py` (`_discover_qc`, add gate after the rnaseq block ~120;
  add imports ~31), `tests/verification/test_run_qc.py` (mirror 119–142).
- **Design (mirror the germline VCF gate, NOT `evaluate_run_qc`):**
  1. A locator `_locate_scrnaseq_qc(run_dir) -> dict[str, dict[str, float]]` that finds the
     aligner artifact(s) under `run_dir` by filename signature — `Summary.csv` (STARsolo),
     `metrics_summary.csv` (Cell Ranger), the simpleaf artifact — derives a **sample id**
     from the enclosing dir, and returns `{sample: parsed_slugs}`.
  2. **Multi-output disambiguation (should-have):** if two aligners' artifacts exist for the
     same sample, prefer deterministically (Cell Ranger > STARsolo > simpleaf) rather than
     merging; document the order. Different samples → separate entries.
  3. In `_discover_qc`, gate `if assay == "scrnaseq":` → `metrics = _locate_...(run_dir)`;
     `results.extend(evaluate(metrics, SCRNASEQ_RULE_PACK))`.
  4. **UNVERIFIED-when-absent:** if a located sample yields no usable slug (empty dict, e.g.
     simpleaf floor), emit one explicit `QCResult(status="unverified", kind="metric")` for
     that sample — mirror `rnaseq_plausibility.py`'s absent-metric branch so a silent
     no-op is impossible. If no artifact at all is found, skip silently (structural QC
     already covers a missing required output) — matching the germline gate's "no VCF" path.
- **RED:** in `test_run_qc.py`: (a) STARsolo fixture dir (healthy) → a `pass` for
  `estimated_cells:<sample>`; (b) near-empty STARsolo → a `fail`; (c) Cell Ranger fixture →
  fraction parsed as 0–1 drives correct status; (d) simpleaf-only / empty → `unverified`,
  never `pass`; (e) **negative:** `_discover_qc(dir, "rnaseq")` and `"variant_calling"` emit
  none of these checks (gate is strictly scrnaseq).
- **GREEN:** implement locator + gate.
- **Validation:** `uv run pytest tests/verification/ -q` then full `uv run pytest -q`
  green; suite ≥ 1157 + new cases.
- **Commit:** `feat(verify): dedicated scrnaseq gate ingests cell-QC into the verdict [M4]`

## Testing strategy

- **Unit:** `test_scrnaseq_metrics.py` — per-parser field mapping, unit normalization
  (`%`→fraction, comma-thousands), absent-field → slug absent, garbage → skipped.
- **Pack:** `test_rule_pack.py` — mito removed, three FAIL bands retained, status math.
- **Integration:** `test_run_qc.py` — gate fires (pass/fail), degrades (unverified),
  disambiguates two aligners, leaves other assays untouched.
- **Command:** `uv run pytest -q` (must stay green); no network, no real pipeline.

## Edge cases & error handling

- Missing artifact → skip silently (structural QC owns "missing output"); parsed-but-empty
  → explicit UNVERIFIED (never a false pass).
- Fraction unit collision (`%` vs 0–1) → explicit per-path test; wrong unit would
  mis-verdict silently.
- Malformed/partial CSV → skip the bad row/field, keep the rest; never fabricate a value.
- Duplicate aligner outputs → deterministic preference, no merge.
- gzip: not expected for these small text QC files; do not add gzip handling unless a
  fixture proves it's needed.
