- **GREEN — implement** `snapshot_history.py`:
  ```python
  """Generic append-only JSONL snapshot store (moat #2 trend persistence).

  Parametrized by the pydantic snapshot model so both the held-out detector trend
  (EvalSnapshot) and the self-heal outcome-match trend (HealSnapshot) share one
  writer/loader. Mirrors eval_history.append_snapshot/load_history, but load skips
  malformed lines too (matches the dashboard reader), so a hand-edited or
  half-written history never crashes --history.
  """
  from __future__ import annotations
  from os import PathLike
  from pathlib import Path
  from typing import TypeVar
  from pydantic import BaseModel, ValidationError

  T = TypeVar("T", bound=BaseModel)

  def append_jsonl(snapshot: BaseModel, path: str | PathLike[str]) -> None:
      p = Path(path)
      p.parent.mkdir(parents=True, exist_ok=True)
      with open(p, "a") as fh:
          fh.write(snapshot.model_dump_json() + "\n")

  def load_jsonl(model_cls: type[T], path: str | PathLike[str]) -> list[T]:
      p = Path(path)
      if not p.exists():
          return []
      out: list[T] = []
      for line in p.read_text().splitlines():
          if not line.strip():
              continue
          try:
              out.append(model_cls.model_validate_json(line))
          except ValidationError:
              continue  # skip a malformed/half-written line; rest is valid
      return out
  ```
  (Catch `ValidationError`; a non-JSON line raises `ValidationError` from
  `model_validate_json` in pydantic v2, so this one except covers both bad-JSON and
  wrong-shape.)
- **Validation:** `uv run pytest tests/test_snapshot_history.py -q` green.
- **Commit:** `feat(history): generic append_jsonl/load_jsonl snapshot store (C6 trend)`

## 4. Testing strategy

All unit, deterministic, `tmp_path` for scratch; the seed tests read the committed files.
No network, no `datetime` use (snapshots are constructed by the test or read from disk).
Command: `uv run pytest tests/test_snapshot_history.py`; gate on full `uv run pytest`.

## 5. Environment & reproducibility

No env vars. The two JSONL files are committed, deterministic artifacts (the durable moat-#2
telemetry). `load_jsonl` tolerating malformed lines is the auditability guarantee — a
corrupt trailing line never hides the valid history.

## 6. Edge cases

- Empty/missing file → `[]` (tested).
- Blank + malformed lines skipped (tested).
- pydantic v2: `model_validate_json` on non-JSON raises `ValidationError` (not
  `JSONDecodeError`) — the single `except ValidationError` covers both. If a runtime shows
  `JSONDecodeError` leaking, widen to `except (ValidationError, ValueError)`.
